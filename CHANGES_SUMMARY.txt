───────────────────────────────────────────────────────────────────────────────
HOW TO USE
───────────────────────────────────────────────────────────────────────────────

The feature is automatically activated in the Reports page. No configuration
needed. The ScannerBadges component:

1. Receives CVE ID and scanner array from ReportsPage
2. Renders colored badges for each scanner
3. Shows "Found by N:" label for multi-scanner detection
4. Provides dropdown menu with external database links
5. Handles all hover interactions and responsive sizing

Component is ready to be reused in other parts of the application.
